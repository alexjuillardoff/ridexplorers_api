Bienvenue sur RIDEXPLORERS API

Cette API REST fournit des informations sur les montagnes russes et les parcs d'attractions extraites de RCDB.

Démarrage rapide

  1. Clonez le projet : git clone https://github.com/alexjuillardoff/ridexplorers_api
  2. Installez les dépendances : npm install
  3. Compilez le code TypeScript : npm run build
  4. Lancez l'API en développement : npm run start:dev

La documentation interactive Swagger est accessible sur /docs une fois le serveur démarré.

Fonctionnalités principales

  • Récupération des montagnes russes et des parcs via des routes REST.
  • Recherche et filtrage par nom ou par région.
  • Lancement des scripts de scraping depuis l'API ou l'interface web.
  • Téléversement de fichiers JSON de données.

Pour un guide pas à pas d'installation et d'utilisation, consultez README.md et GUIDE_DEBUTANT.md dans le dépôt GitHub.
